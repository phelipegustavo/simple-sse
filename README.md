# Simple Server-Sent Events in Node.js + React.js

Demo using SSE with node.js and react

## How to run

### Client 

```
yarn
yarn start
```

### Server

```
cd server
yarn
yarn start
```

### Post a new fact

```
./post-fact.sh
```
